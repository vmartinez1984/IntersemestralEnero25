Clase 05

El despliegue de software se refiere al proceso de poner una aplicación o servicio en un entorno de producción, haciendo que esté disponible para los usuarios finales. Algunos métodos comunes de despliegue incluyen el manual, mediante scripts y continuo. Implica probar el software en su ambiente operacional final, empacar el software para la entrega, distribuir el software, instalar el software, entrenar a los usuarios finales y convertir las bases de datos anteriores para la carga de datos.
